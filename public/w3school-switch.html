Select a Fruit

Enter a fruit name to see its color:

• This example uses the switch statement to match a fruit name with its color.

Find the Grade

Enter a score (0-100) to see the grade:

• This example uses the switch statement to determine a grade based on score ranges.
• For example, scores 90 and above are "A", 80-89 are "B", and so on.
• Any score outside 0-100 results in an invalid message.

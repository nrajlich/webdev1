How It Works:

  1. User Input: The user enters a temperature in Celsius in the input box.
  2. celsiusToFahrenheit Function:
    • Takes a Celsius temperature as input.
    • Converts it to Fahrenheit using the formula:
      Fahrenheit = (Celsius × 9/5) + 32
  3. Result Display: After clicking the button, the converted Fahrenheit value is displayed below.

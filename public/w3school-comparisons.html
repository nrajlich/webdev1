JavaScript Comparisons

Equality (==) Comparison

Check if two numbers are equal:

The equality operator == checks if two values are equal. It does not consider data type.

Strict Equality (===) Comparison

Check if two values and their types are equal:

The strict equality operator === checks if two values and their data types are equal.

Greater Than (>) Comparison

Check if one number is greater than another:

The greater than operator > checks if the first value is larger than the second.

Less Than or Equal (<=) Comparison

Check if one number is less than or equal to another:

The less than or equal operator <= checks if the first value is smaller than or equal to the second.

JavaScript Let Variables

In this example, let is used to declare variables a, b, and c.

The let keyword means these variables can be reassigned later if needed.

JavaScript Const Variables

In this example, const is used to declare variables x, y, and z.

The const keyword means these variables are constants and cannot be reassigned after their initial declaration.

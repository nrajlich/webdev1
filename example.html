Work

Fell Swoop is a digital agency with a focus on user experience. I was a partner and lead designer on a wide range of design and research projects.

Expertise

User Experience Design

User profiles, personas, task analysis, scenarios, user flows, user stories, experience mapping.

Interaction Design

UI design, paper prototyping, wireframe prototyping, interactive prototyping, high fidelity wireframes.

Information Architecture

Navigation design, labeling, content inventory, taxonomy, metadata, tagging, search design.

User Research

Contextual inquiry, user interviews, user surveys, card sort studies, mental models.

Usability Testing

Test plan, participant screener, session script, remote testing, in-person moderation, findings analysis, recommendations report.

Expert Review

Heuristic analysis, cognitive walkthrough, issue identification, severity ranking, recommendations report.

My process is based on established user-centered design methods, but is adjusted and optimized for each situation and client.
